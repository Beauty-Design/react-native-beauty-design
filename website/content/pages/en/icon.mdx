#### Icon Props
More props in [react-native-vector-icons](https://github.com/oblador/react-native-vector-icons)

| Attribute | Type                    | Accepted values                                                  | Description            | Default     |
|-----------|-------------------------|------------------------------------------------------------------|------------------------|-------------|
| **color** | `ThemeColor` `string`   | [ThemeColor](/docs/default-theme)                                | Change icon color      | `text`      |
| **name**  | `string`                | [IconName](https://oblador.github.io/react-native-vector-icons/) | Name of icon to render | `undefined` |
| **size**  | `number`                | Number                                                           | Size icon              | 18          |
| **type**  | `IconProvider` `string` | [IconProvider](#icon-provider)                                   | Icon provider type     | `material`  |
